我一直坚持写博客的原因

发布日期:   2019-12-20
更新日期:   2020-05-29
文章字数:   1.2k
阅读时长:   4 分

我们信誓旦旦，面对死亡！

一

前段时间无意间看了 laike9m 写的一篇随笔《对抗消失的勇气》，主要讲的是内容创作者的动机主要源于人类想要表达的欲望。然而，人为什么要表达？可能很大程度上是为了确认自身的存在。其中有两点触动了我的内心，因为基本和我的想法是一致。之前一直想写一篇文档，讲述我一直写博客的原因和本源出发点。刚刚好，看了这篇随笔之后，我也想记录点什么。

二

以前和朋友聊天，他说死亡最可怕的地方不是痛苦，而是在于死了之后什么都没有留下，甚至连痛苦都感受不到。而活着的人，如果没办法感受到任何事，其实和死了也差不多。我一直坚持写博客，正是因为我希望通过这种方式来感受和反馈信息给这个世界。笛卡尔说过“我思故我在”，他通过自己正在思考存在这件事确认自己的存在。这种证明对于普通人还是太抽象了，因此我们需要一些更简单、更直接、更有形的东西。而对于我来说，这件东西就是写博客，这样能够告诉这个世界我来过或者我曾经来过。

之前有人问我为啥要再开博客而不是写公众号，我想今天大规模删帖的事情已经给出很明确的答案了。我不认为流量垄断、信息封锁和信息审查是互联网该有的样子，互联网的创造和普及是我心目中人类有史以来最伟大的公益事业，我希望互联网能够持续公开透明可检索，而不是变成一个个商业巨头的流量池。

之前在 Twitter 上面了解到，Joe Armstrong 将自己的博客迁移到 tiddlywiki 中去，用于保存自己的所有记录。现在，虽然他人已经去世了，但是还是有人不断在下面留言和评论。即使他的肉体已经死亡，只要作品还在，从某种意义上来讲，他这个人就还存在。对于我来说，正是如此！我们可以想想，从自己出生到死亡，短短几十年时间，而且往往有很多人因为意外或者疾病等原因，还达不到这个时间级别。而几十年时间，对应这个世界来说，是不是显得很微小。正如我们从摩天大楼，俯身看下面的人一样，小小的人如同蚂蚁一般。因为人本来就是一个微小的存在，任何人的死亡或者陨落，对于他/她周围的人、这个国家、这个世界来说到底有多少意义呢？

三

我也不记得从什么时候开始，在夜深人静的时候，我总会不断的在问我自己一个问题，那就是 人活着的意义到底是什么？ 我一直没有答案，或者说我自己找到的答案都很说服我自己。

有一天，我把这个问题说给了圆圆。她说，这个问题其实和问我 “生孩子有什么意思呢？” 是一样的。我们把孩子生下来之后，需要给他/她买奶粉、供给他/她上学、给他/她结婚，想想是不是很亏呢？如果等他/她长大之后埋怨你没有给他/她一个更好地生活条件，想想是不是很恼火呢？。咳咳，你应该思考的是：你生孩子不是要给孩子带来什么，而是孩子给我们带来什么！

人活着不是世界给我们什么意义，而是我们给世界留下了什么意义！

人活着，总是想留下一些痕迹的，用来证明自己曾经来过这个世界。然而，事实上 99.999% 的普通人就这么被历史遗忘了——曾经是这样，以后也会是这样。人有两次死亡，第一次是肉体，第二次是被人忘记。 我忘记这句话是谁说的了，但现在用 写博客和 GitHub 来努力回避第二次。我相信，几百年之后 GitHub 将成为世界上最大的数字公墓，注册用户大部分都已去世，然而个人主页、项目，commit 历史还在述说着他们生前做过的事——就比如 Joe 的博客。

卒！
