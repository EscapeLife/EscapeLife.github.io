自定义VSCode工具快捷键

更新日期:   2021-03-04
文章字数:   1.7k
阅读时长:   6 分

要使工具用起来顺手，还是需要自己定义一些快捷键！

1. 界面控制相关

红藕香寒翠渚平，月笼虚阁夜蛩清，塞鸿惊梦两牵情。 > 宝帐玉炉，残麝冷，罗衣金缕暗尘生，小窗孤烛泪纵横。

  • [1] 常用的界面操作
编号 快捷键 对应说明
1 Command + p 快速打开文件
2 Command + Shift + p 显示命令面板
3 Command + k + m 更改语言模式
4 Command + e + l 切换项目代码
5 Command + , 打开用户设置
6 Command + - 缩小缩小
7 Command + + 放大界面
8 Command + Ctrl + f 进入/退出全屏
9 Command + Shift + ? 界面搜索常见快捷键
10 Command + k; Command + t 修改颜色主题
11 Command + k; Command + s 打开键绑定
12 Command + k; Command + k 修改键绑定
13 Command + Option + k 录制按键
14 Command + k; Command + Backspace 删除键绑定
  • [2] 左侧导航栏操作
编号 快捷键 对应说明
1 Command + 0 聚焦到侧边栏
2 Command + b 显示/隐藏侧边栏
3 Command + Shift + e 显示资源管理器
4 Command + Shift + f 显示搜索框
5 Command + Shift + h 全局搜索替换
6 Command + Shift + j 显示/隐藏高级搜索
7 Command + Shift + d 显示调试面板
8 Command + Shift + g 显示 Git 面板
9 Command + Shift + x 显示插件面板
10 Command + k + z 切换到禅模式
  • [3] 下侧导航栏操作
编号 快捷键 对应说明
1 Command + j 切换面板
2 Ctrl + 右撇 显示终端
3 Command + Shift + m 显示问题视图
4 Command + Shift + u 显示输出面板
5 Command + Shift + y 显示调试控制台

2. 代码索引相关

惆怅经年别谢娘，月窗花院好风光，此时相望最情伤。青鸟不来传锦字，瑶姬何处琐兰房，忍教魂梦两茫茫。

  • [1] 查看代码引入和定义
编号 快捷键 对应说明
1 Ctrl + m 查看引用
2 Command + m 跳转定义
3 Command + e + p 显示整个项目引用的地方
4 Command + e + d 侧窗口打开定义
5 Command + t 全局搜索函数定义和引用
6 Command + shift + . / Command + shift + o 切换代码导航视图
  • [2] 代码跳转和缩进相关
编号 快捷键 对应说明
1 Ctrl + g 转到行
2 Command + . 快速修复
3 Command + / 切换行注释
4 Command + [ 行缩进
5 Command + ] 行缩进
6 Command + Shift + \ 匹配的括号
7 Option + Shift + o 排序 import 语句
8 Command + k; Command + j 展开全部代码块
9 Command + k; Command + l 折叠当前代码块
10 Command + k; Command + a 折叠所有代码块
11 Command + k; Command + / 折叠所有注释块
12 Command + k; Command + 1-8 折叠第 1-8 层嵌套

3. 查找替换相关

梁燕双飞画阁前，寂寥多少恨，懒孤眠。晓来闲处想君怜，红罗帐，金鸭冷沉烟。

编号 快捷键 对应说明
1 Command + f 查找当前单词
2 Command + Shift + f 全局查找当前单词
3 Enter 查找下一个
4 Shift + Enter 查找上一个
5 Option + Enter 选中所有匹配项
6 Command + e + f 替换当前单词
7 Command + Shift + h 全局替换当前单词
8 Command + d 将下一个匹配项选中

4. 编辑页面相关

七夕年年信不违，银河清浅白云微，蟾光鹊影伯劳飞。每恨蟪蛄怜婺女，几回娇妒下鸳机，今宵嘉会两依依。

  • [1] 文章编辑相关操作
编号 快捷键 对应说明
1 Command + e + x 裁剪尾随空格
2 Command + e + j 合并行内容
3 Option + Shift + f 格式化文档
4 Command + k; Command + f 格式化文档
5 Command + Shift + t 重新打开已经关闭的 Tab 页面
  • [2] 页面跳转相关操作
编号 快捷键 对应说明
1 Command + w 关闭页面窗口
2 Command + shift + w 关闭整个项目窗口
3 Command + k + f 关闭工作区域窗口
4 Ctrl + Tab 切换下一个 Tab 页面
5 Ctrl + Shift + Tab 切换上一个 Tab 页面
6 Command + shift + [ 打开左侧 Tab 页面
7 Command + shift + ] 打开右侧 Tab 页面
8 Command + \ 编辑器分屏
9 Command + k; Command + \ 下方新建一个 tab 分组
10 Command + 1 切换到第一分组
11 Command + 2 切换到第二分组
12 Command + 3 切换到第三分组
13 Command + Option + 0 切换垂直/水平布局分组
14 Command + k; Command + up 切换到上一分组
15 Command + k; Command + down 切换到下一分组
16 Command + k; Command + left 切换到左一分组
17 Command + k; Command + right 切换到右一分组

5. 问题调试相关

万株枯槁怨亡隋，似吊吴台各自垂。好是淮陰明月里，酒楼横笛不胜吹。

编号 快捷键 对应说明
1 Command + d + s 调试开始执行
2 Command + d + r 不调试开始执行
3 Command + d + b 添加函数断点
4 Command + d + d 删除所有断点
5 Command + d + l 当前行添加/删除断点

6. 终端控制相关

阊门风暖落花干，飞遍江城江城雪不寒。独有晚来临水驿，闲人多凭赤栏干。

编号 快捷键 对应说明
1 Ctrl + 右撇 显示终端
2 Command + \ 终端拆分
3 Command + Option + 0 切换垂直/水平布局
4 Command + f 查找所需内容
5 Command + k 清空所有内容
6 Command + ← 移动到行首
7 Command + → 移动到行尾
8 Command + ↑ 滚动到上一条命令
9 Command + ↓ 滚动到下一条命令
10 Command + Option + ← 聚焦到左侧窗格
11 Command + Option + → 聚焦到右侧窗格
12 Command + Option + ↑ 聚焦到上侧窗格
13 Command + Option + ↓ 聚焦到下侧窗格
14 Command + e + b 最大化面板大小

7. 移动选中相关

景陽钟动宫莺啭，露凉金殿。轻飙吹起琼花旋，玉叶如剪。

编号 快捷键 对应说明
1 Option + 点击 插入多个光标
2 Option + Shift + 拖动鼠标 选中代码块
3 Command + Up 跳转至文件开头
4 Command + Down 跳转至文件结尾
5 Option + Up 将当前行向上移动行
6 Option + Down 将当前行向下移动行
7 Ctrl + PgUp 按行向上滚动
8 Ctrl + PgDown 按行向下滚动
9 Command + PgUp 按屏向上滚动
10 Command + PgDown 按屏向下滚动
11 Command + Enter 下一行插入内容
12 Command + Shift + Enter 上一行插入内容

8. 其他高级相关

莺啼燕语芳菲节，瑞庭花发。昔时欢宴歌声揭，管弦清越。

编号 快捷键 对应说明
1 Option + Shift + d 通过 Rsync 下载 Vscode 配置
2 Option + Shift + u 通过 Rsync 上传/更新 Vscode 配置
3 Option + m 显示 Markdown 菜单
4 Command + k + v 打开侧边预览
5 Option + Shift + Enter 全屏显示 Markdown 文档
6 Command + Option + p 打开项目管理
7 Option + Shift + b 打开浏览器预览
8 Command + e + u 更改文件编码
9 Command + Option + m 格式化 Json 文档
10 Option + m 反格式化 Json 文档
